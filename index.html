Interactive Topic Modelling Visualization

This GitHub Pages site hosts an interactive pyLDAvis topic model view. You can explore topic distances, most relevant terms, and term saliency directly in the browser.

About this page

This page is ready to publish on GitHub Pages as a standalone website.

  • Left panel: topic circles and their distances
  • Right panel: top terms for each topic
  • Use the relevance slider to inspect broader vs. more topic-specific words
